🎨 Canva Editor Template System

📋 Features Implemented

  • Template Export - Save current editor state as JSON
  • Template Import - Load templates from JSON files or URLs
  • Parameter Substitution - Replace text and image placeholders
  • Template Customization - Real-time preview of changes
  • API Endpoints - RESTful API for template management
  • React Components - Ready-to-use UI components

🔧 API Endpoints

POST /api/template/export
{ "templateName": "My Template", "description": "Template description", "pages": [...], "author": "Author Name", "tags": ["tag1", "tag2"] }
POST /api/template/import
{ "templatePath": "./json/sample-template.json", "textParameters": { "title": "Custom Title", "subtitle": "Custom Subtitle" }, "imageParameters": { "backgroundImage": "https://example.com/image.jpg" } }
GET /api/template/{templateName}
GET /api/template/sample-template?textParameters={"title":"Custom Title"}&imageParameters={"backgroundImage":"https://example.com/bg.jpg"}

💻 Usage Examples

Export Template

import { createTemplateFromEditor } from 'canva-editor/utils/templateExport'; const templateData = createTemplateFromEditor( currentPages, 'My Template', { description: 'A beautiful template', author: 'John Doe', tags: ['business', 'modern'], download: true } );

Import Template with Parameters

import { loadTemplateFromFile, importTemplate } from 'canva-editor/utils/templateImport'; const templateData = await loadTemplateFromFile(file); const customizedPages = importTemplate(templateData, { templatePath: './template.json', textParameters: { title: 'Welcome!', subtitle: 'Get started today' }, imageParameters: { backgroundImage: 'https://example.com/bg.jpg' } });

React Components

import { TemplateManager, TemplateCustomizer } from 'canva-editor/components/template'; // Template Manager Component <TemplateManager onClose={() => setShowManager(false)} /> // Template Customizer Component <TemplateCustomizer templateData={templateData} onApply={(customizedPages) => { // Apply to editor customizedPages.forEach((page, index) => { actions.setPage(activePage + index, page); }); }} />

🧪 Testing

To test the template system:

  1. Start the API server: cd canva-editor/api && npm start
  2. Run the test script: node test-template-system.js
  3. Use the React components in your editor
